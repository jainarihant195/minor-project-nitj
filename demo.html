Gate Pass Rules

Rule 1

A gate pass can only be used once.

Rule 2

No gate pass will generated after 8PM.

Rule 3

Your gate pass should be approved by hostel admin before you go out.

Rule 4

You have return to hostel before 9PM(if not on leave).
